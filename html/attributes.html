Lesson 2 - What Are HTML Attributes?

Definition

  • Attributes are values placed inside the opening tag of an HTML element.
  • They provide additional information or specify behavior for the element.
  • Basic syntax: <element attribute="value"></element>

Common HTML Attributes

  • href → Specifies the URL of a link
    Example: <a href="https://www.example.com">Visit</a>
  • src → Specifies the source of an image
    Example: <img src="image.jpg" alt="A beautiful image" />
  • alt → Provides descriptive text for images (important for accessibility).
  • checked → Boolean attribute used in checkboxes to mark them as checked by default.
    Example: <input type="checkbox" checked />
  • Other common boolean attributes: disabled, readonly, required

Importance of Attributes

  • Help make web content more interactive and accessible.
  • Allow customization of elements to suit different functionalities.
