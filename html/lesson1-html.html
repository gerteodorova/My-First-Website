HTML on the Web

Lesson 1 - What is HTML?

Definition

  • Stands for Hypertext Markup Language
  • Used to structure and display content on web pages
  • Represents elements like paragraphs, headings, links, images, videos

HTML Elements & Tags

  • Most elements have an opening tag and a closing tag
  • Example: <p>Hello</p>
  • Opening and closing tags:
    • Opening tag: <p>
    • Closing tag: </p> (contains a forward slash /)
  • Void elements (do not require a closing tag)
    • Example: <img>
