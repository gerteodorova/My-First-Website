HTML on the Web

Lesson 1 - What is HTML?

Definition

  • Stands for Hypertext Markup Language
  • Used to structure and display content on web pages
  • Represents elements like paragraphs, headings, links, images, videos

HTML Elements & Tags

  • Most elements have an opening tag and a closing tag
  • Example: <p>Hello</p>
  • Opening and closing tags:
    • Opening tag: <p>
    • Closing tag: </p> (contains a forward slash /)
  • Void elements (do not require a closing tag)
    • Example: <img>

Attributes in HTML

  • Used to modify behavior of elements
  • Example: <img src="image-location" />
  • Common attributes:
    • src (source) → specifies image location
    • alt (alternative text) → provides image description

HTML, CSS, and JavaScript: How They Work Together

  • HTML → Content & Structure (like the foundation of a building)
  • CSS → Styling (like interior & exterior design)
  • JavaScript → Interactivity (like electrical & water systems)

Lesson 2 - What Are HTML Attributes?

Definition

  • Attributes are values placed inside the opening tag of an HTML element.
  • They provide additional information or specify behavior for the element.
  • Basic syntax: <element attribute="value"></element>

Common HTML Attributes

  • href → Specifies the URL of a link
    Example: <a href="https://www.example.com">Visit</a>
  • src → Specifies the source of an image
    Example: <img src="image.jpg" alt="A beautiful image" />
  • alt → Provides descriptive text for images (important for accessibility).
  • checked → Boolean attribute used in checkboxes to mark them as checked by default.
    Example: <input type="checkbox" checked />
  • Other common boolean attributes: disabled, readonly, required

Importance of Attributes

  • Help make web content more interactive and accessible.
  • Allow customization of elements to suit different functionalities.

Lesson 3 - Using the Link Element

  • The <link> element is used to link external resources like stylesheets and icons.
  • Basic syntax: <link rel="stylesheet" href="./styles.css" />
  • rel: Specifies the relationship between the document and the linked resource.
  • href: Specifies the URL of the external resource.
  • Example usage inside the <head> section: 
        <head>
      <meta charset="UTF-8" />
      <meta name="viewport" content="width=device-width, initial-scale=1.0" />
      <title>Example</title>
      <link rel="stylesheet" href="./styles.css" />
    </head>
  • Can also be used to link Google Fonts: 
        <link rel="preconnect" href="https://fonts.googleapis.com" />
    <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin />
    <link href="https://fonts.googleapis.com/css2?family=Playwrite+CU:wght@100..400&display=swap" rel="stylesheet" />
  • Used to add a favicon (site icon):
    <link rel="icon" href="favicon.ico" />
