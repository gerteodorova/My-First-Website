What Is HTML, and What Role Does It Play on the Web?

Definition of HTML

  • Stands for Hypertext Markup Language
  • Used to structure and display content on web pages
  • Represents elements like paragraphs, headings, links, images, videos

HTML Elements & Tags

  • Most elements have an opening tag and a closing tag
  • Example: <p>Hello</p>
  • Opening and closing tags:
    • Opening tag: <p>
    • Closing tag: </p> (contains a forward slash /)
  • Void elements (do not require a closing tag)
    • Example: <img>

Attributes in HTML

  • Used to modify behavior of elements
  • Example: <img src="image-location" />
  • Common attributes:
    • src (source) → specifies image location
    • alt (alternative text) → provides image description

HTML, CSS, and JavaScript: How They Work Together

  • HTML → Content & Structure (like the foundation of a building)
  • CSS → Styling (like interior & exterior design)
  • JavaScript → Interactivity (like electrical & water systems)
